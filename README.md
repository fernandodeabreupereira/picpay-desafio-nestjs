# **Desafio Backend PicPay**

## Sobre
Projeto desenvolvido para treinar habilidades, baseado no desafio de backend do picpay (https://github.com/PicPay/picpay-desafio-backend)

## Instalação

```bash
$ npm install
```

## Rodando o app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Testes

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
